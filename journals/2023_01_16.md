- ### Installer acl
	- apt install acl
-
- ### monter le disque avec acl activé :
	- `/dev/sda1  /      ext4    acl,defaults                0 1`
-
- ### Ajouter des droits à un groupe :
	- `setfacl -m g:1001:rx file.txt`
-
- ### Information sur un utilisateur :
	- `id`
-
- ### Changer le mask ACL :
	- `setfacl -m mask:rwx dir`
